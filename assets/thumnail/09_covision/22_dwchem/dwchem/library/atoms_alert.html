Alert

오류를 안내하는 한줄짜리 alert 영역입니다. 한줄짜리 얼럿으로 사용하기 위해서는 alert_basic class가 필요합니다.
data-type 값으로 alert의 종류를 구분합니다.

modal에 cui_alert 컴포넌트를 활용하는데 이때, alert_basic대신 alert_modal클래스를 사용합니다. 참고

This is an info alert — check it out!

This is a success alert — check it out!

This is an error alert — check it out!

This is a warning alert — check it out!

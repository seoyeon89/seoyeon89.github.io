Input

Text Input

input은 한 줄의 텍스트를 입력 할 수 있는 컴포넌트입니다.

버튼과 결합한 구조가 필요한 경우(ex. datePicker) 링크를 참고하여 주시기 바랍니다.

input은 data-ui-size값을 조정해서 size를 변경할 수 있고, 지정된 사이즈 외에 사용을 하고 싶은 경우는 style="width:(원하는값)px"를 태그의 inline style 로 추가하여 사용하면 됩니다.

data-ui-size 값 width값
short 60px
small 160px
basic 200px
medium 240px
large 320px

Textarea

Textarea는 여러 줄의 일반 텍스트 편집 컨트롤입니다. 입력 영역 크기는 세로로만 변경 가능합니다.
